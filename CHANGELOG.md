## [v0.1.0] - 2026-04-26

### Highlights

- DataFrame library for Go with XLSX export support

### Added

- Core `Frame` type for tabular data with named columns ([`0c2cfa4`](https://github.com/plexusone/omniframe/commit/0c2cfa4))
- Core `Column` type with automatic type inference (string, int64, float64, bool, time) ([`0c2cfa4`](https://github.com/plexusone/omniframe/commit/0c2cfa4))
- `FrameSet` type for multi-sheet workbook support ([`0c2cfa4`](https://github.com/plexusone/omniframe/commit/0c2cfa4))
- `FromColumns()` and `FromRows()` constructors for creating frames ([`0c2cfa4`](https://github.com/plexusone/omniframe/commit/0c2cfa4))
- Frame operations: `Head()`, `Tail()`, `Select()`, `Clone()` ([`0c2cfa4`](https://github.com/plexusone/omniframe/commit/0c2cfa4))
- Column formatting with Excel number formats (currency, percent, date, etc.) ([`0c2cfa4`](https://github.com/plexusone/omniframe/commit/0c2cfa4))
- XLSX export via `WriteXLSX()` for Frame and FrameSet ([`df0fd79`](https://github.com/plexusone/omniframe/commit/df0fd79))
- Column width support for Excel output ([`df0fd79`](https://github.com/plexusone/omniframe/commit/df0fd79))
